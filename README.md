# Astro Trading

A lightweight static website for Astro Trading — marketing/static site built with plain HTML, CSS and JavaScript.

Repository: https://github.com/FaisalRahmanGit/astro-trading.git

## Summary

This project is a small multi-page static site containing:

- `index.html` — Home page
- `about-us.html` — About page
- `contact.html` — Contact form / details
- `faq.html` — Frequently asked questions
- `terms-of-service.html` — Legal / TOS
- `assets/` — CSS, JavaScript, images and fonts

## Features

- Responsive layout using CSS (includes `bootstrap.min.css` and `main.css`)
- Interactive elements with plain JavaScript (`script.js`)
- Image and font assets located in `assets/`
- Simple, easy-to-edit HTML structure for quick content updates

## File Structure

- `index.html`
- `about-us.html`
- `contact.html`
- `faq.html`
- `terms-of-service.html`
- `assets/`
  - `css/` — `bootstrap.min.css`, `main.css`, `swiper.min.css`
  - `js/` — `jquery-3.5.1.min.js`, `bootstrap.min.js`, `script.js`, `swiper.min.js`
  - `images/` — site images
  - `fonts/` — custom fonts
