# Astro Trading

Astro Trading is a responsive, multi-page static marketing website built using plain HTML, CSS, and JavaScript.  
The project demonstrates frontend fundamentals, clean UI layout, and integration of popular animation and UI libraries.

This repository is intended as a **portfolio and academic project**, showcasing practical web development skills rather than a production trading platform.

---

## Project Overview

The website represents a fictional trading community landing page and includes multiple informational sections such as services, testimonials, FAQs, and contact details. It focuses on visual presentation, responsiveness, and user-friendly structure.

---

## Features

- Fully responsive multi-page layout  
- Clean and easy-to-edit HTML structure  
- Smooth scroll-based animations using AOS and Animate.css  
- Interactive testimonial slider using Swiper.js  
- Bootstrap-based grid system for layout consistency  
- Organized assets (CSS, JS, images, fonts)

---

## Tech Stack

- **HTML5**
- **CSS3**
- **Bootstrap**
- **JavaScript (Vanilla)**
- **jQuery**
- **Swiper.js**

## File Structure

This project is a small multi-page static site containing:
- `index.html` — Home page
- `about-us.html` — About page
- `contact.html` — Contact form / details
- `faq.html` — Frequently asked questions
- `terms-of-service.html` — Legal / TOS
- `assets/` — CSS, JavaScript, images and fonts
    - `css/` — `bootstrap.min.css`, `main.css`, `swiper.min.css`
    - `js/` — `jquery-3.5.1.min.js`, `bootstrap.min.js`, `script.js`, `swiper.min.js`
    - `images/` — site images
    - `fonts/` — custom fonts
